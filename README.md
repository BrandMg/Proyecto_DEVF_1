# Página Web sobre Nebulosas

Este proyecto es una práctica básica de HTML cuyo objetivo es crear y publicar una página web sencilla utilizando **GitHub** Pages. La temática elegida para la página es las nebulosas, objetos astronómicos formados por gas y polvo interestelar.


## Tecnologías Utilizadas
* HTML5
* Git y GitHub para control de versiones
* GitHub Pages para el despliegue


## Objetivo de la práctica
El objetivo principal es reforzar el uso de HTML básico, control de versiones y despliegue en GitHub Pages.


## Características del proyecto
La página web incluye los siguientes elementos:

1. Estructura semántica básica con etiquetas: html, head, body, title, meta.
2. Encabezados (h1, h2) para organizar la información.
3. Párrafos descriptivos usando p.
4. Enlaces de navegación mediante la etiqueta a.
5. Imágenes representativas de nebulosas usando img.
6. Listas (ul) para organizar contenido.
7. Formulario básico con form, label, input y textarea.
8. Navegación interna mediante anclas (#seccion).


## Despliegue en GitHub Pages
Para visualizar la página en línea:

1. En tu repositorio ve a Settings.
2. Desplázate a la sección Pages.
3. En "Source", selecciona la rama main y la carpeta /root.
4. Guarda los cambios.
5. GitHub generará un enlace similar a: https://tuusuario.github.io/nombre-del-repositorio/ (Para esta pagina web sera: https://brandmg.github.io/Proyecto_DEVF_1)
6. Al abrir este enlace se desplegara la pagina correspondiente.
