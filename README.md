# Página Web sobre Nebulosas

Este proyecto es una práctica básica de HTML cuyo objetivo es crear y publicar una página web sencilla utilizando GitHub Pages. La temática elegida para la página es las nebulosas, objetos astronómicos formados por gas y polvo interestelar.


# Tecnologías Utilizadas
-HTML5
-Git y GitHub para control de versiones
-GitHub Pages para el despliegue


# Objetivo de la práctica
El objetivo principal es reforzar el uso de HTML básico, control de versiones y despliegue en GitHub Pages.
